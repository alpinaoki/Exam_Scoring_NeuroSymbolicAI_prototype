### 1. 数学系科目における手書き答案の自動採点 (JSET2025-2-D12)

**論文名：** 生成AIを用いた数学系科目における演習問題の採点および添削手法の開発と学生へのフィードバック
**著者：** 西出崇（小樽商科大学）、中村文彦（北見工業大学）
この研究は、大学の数学授業における**手書きの演習問題**を生成AI（OpenAI API）を用いて採点・添削し、学生へフィードバックする仕組みの実践報告です。

 **目的と背景：**
 多人数講義において、個々の学生の理解度に応じた形成的評価や個別フィードバックを行うには多大な労力を要するため、生成AIによる自動化を検証する 。
 理工系学生の手書き答案（数式やグラフ）を画像として読み取り、採点・添削を行うシステムの開発を目指した 。
 **システムの仕組み：**
 学生の答案画像を切り出し、正答や詳細な採点基準（部分点基準など）を記したPDFと共にAIモデル（OpenAI API）に入力する 。
 出力結果（採点、誤りの指摘、アドバイス）を加工し、学生へのフィードバック用PDFを生成する 。
 **検証結果（AIモデルの比較）：**
 採点精度（教員の採点との一致率）を比較した結果、汎用モデル（GPT-4o, GPT-4.1）の一致率は低かった（26〜43%程度） 。
 一方で、推論モデルである **o1-mini は70.2%と高い一致率**を示し、模範解答と異なる解法（別解）や式の変形も正しく判定できた 。
 「数学に強いモデル」を利用すれば、追加学習なしでもかなりの精度と品質が担保されることが示唆された 。
 **学生の反応とコスト：**
 学生の約80%が「採点基準に合っている」と回答し、約86%がフィードバックを「わかりやすい」と評価した 。
 コストは学生100人分で2〜3ドル未満と安価であり、実用的である 。

### 2. 中学校国語科における記述式問題の採点支援 (2024_kyusyu_okinawa_a09)

**論文名：** AIを活用した記述式問題の採点支援 -ChatGPTによるシステム提案と評価-
**著者：** 瀧澤陽華、小渡悟（沖縄国際大学）
この研究は、中学校の国語における**自由記述問題（作文など）**の採点負担を軽減するため、ChatGPT（Custom GPTs）を活用した採点システム「採点くん」を開発・評価したものです。

**目的と課題：**
 既存の採点アプリはマークシート形式などが主流で、自由記述に対応したものが少ない 。
 国語の記述問題は多様な観点が必要で教員の負担が大きいため、ChatGPTを用いた効率化を目指す 。
 **提案システム「採点くん」：**
 ChatGPTのカスタムモデル（GPTs）を使用し、深津式プロンプト（命令・制約条件などを明確化する手法）を採用 。
 学習指導要領に基づくルーブリック（評価基準）を知識としてAIに与え、文脈に即した採点を行わせる 。
 **検証結果：**
**利点：** 漢字や文法の正確性に対する指摘は精度が高かった。また、人力と比較して圧倒的に速く採点できた 。 
**課題：** 長文の論理性の一貫性評価に欠ける場合があったほか、文字数制限（「150字以上」など）を満たしているのに誤認するケースが見られた 。
 現状では完全自動化は難しいが、部分的な支援ツールとしては有用であると結論付けている 。

### 3. 採点プロセスの概念図 (default.pdf)

**資料内容：** 「答案収集」から「UI表示」までのフロー
この資料は、AIを用いた採点システムの処理フローをメモ書きしたもののようです。上記2つの論文のシステム構成と共通する概念が含まれています。
* **プロセス：**
1. 
**答案収集(A)：** データ送信 。
2. 
**構造分析(B)：** 要素分解、論理構造の理解 。
3. 
**判定・生成：** 内容の確率（確信度）計算、論理破綻の検知、加点・減点の算出 。
4. 
**UI表示(D)：** 「この部分がおかしい」という指摘や改善方法、コメントを学生に提示する 。

### 総合的なまとめ

これら3つの資料は、**「生成AIは教育評価（採点・添削）において、教員の補助ツールとして実用段階に入りつつある」**という共通の結論を示しています。

* **数学（数式・論理）：** 最新の推論モデル（o1-mini等）を使えば、手書きの数式理解や別解判定も含めて非常に高い精度が出る。
* **国語（記述・文章）：** 文法や漢字チェックには強いが、文字数カウントや長い論理構成の評価にはまだ課題が残る。
* **共通点：** どちらの研究も、最終的な成績評価をAIに任せるのではなく、**「形成的評価（学習途中のフィードバック）」や「教員の業務支援」**として有効であると位置づけています。
